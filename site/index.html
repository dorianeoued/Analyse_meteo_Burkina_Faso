Source & Qualité

Les Données

Les données météorologiques couvrent 33 ans (1990–2023) pour 6 villes représentatives du Burkina Faso, du Sahel au Sud soudanien.

📡 Source : NASA POWER (Prediction Of Worldwide Energy Resources)

L'API NASA POWER fournit des données climatiques journalières dérivées des observations satellitaires et des réanalyses atmosphériques. Quatre paramètres ont été téléchargés : précipitations corrigées (PRECTOTCORR), température maximale (T2M_MAX), température minimale (T2M_MIN) et humidité relative (RH2M). Communauté agricole (AG), résolution journalière, format JSON.

⚠️ Qualité des données

  • Valeurs manquantes gérées : NASA POWER encode les données manquantes avec la valeur sentinelle -999.0. Celles-ci ont été remplacées par NaN (pandas) avant tout calcul, afin d'éviter des biais dans les statistiques.
  • Valeurs extrêmes non traitées : Aucun filtre sur les valeurs aberrantes (ex. précipitations > 300 mm/jour) n'a été appliqué. Ces événements extrêmes rares, bien que physiquement plausibles au Sahel lors de lignes de grains intenses, peuvent légèrement biaiser les moyennes annuelles.
  • Données satellitaires vs stations : Les données NASA POWER sont issues de modèles de réanalyse et peuvent différer des mesures in-situ pour des événements locaux très intenses.

Six techniques d'analyse de données utilisées dans ce projet, appliquées avec Python (pandas, numpy, matplotlib).

Nettoyage des données
Remplacement des valeurs sentinelles -999.0 par NaN (pandas) pour neutraliser les données manquantes sans biaiser les calculs statistiques.

Rééchantillonnage temporel
Agrégation des données journalières vers des échelles mensuelles puis annuelles via groupby et resample, en adaptant la fonction d'agrégation (somme, moyenne) au paramètre.

Analyse de tendance
Régression linéaire par moindres carrés (np.polyfit) sur 33 ans pour quantifier la direction et l'amplitude des évolutions de précipitations et de températures.

📊
Calcul d'anomalies
Écart en pourcentage de chaque année par rapport à la moyenne de référence de la période complète. Permet d'identifier les années extrêmes indépendamment des volumes absolus.
Climatologie
🌧️
Détection d'événement
Début de saison des pluies défini par le critère agronomique : cumul sur 3 jours consécutifs ≥ 20 mm après le 1er avril (jour julien 90). Détecté par fenêtre glissante sur la série journalière.
Agronomie
📋
Statistiques descriptives
Calcul de la moyenne, de l'écart-type, du minimum et du maximum par ville et par période. Permet la comparaison du gradient climatique Nord-Sud et de la variabilité interannuelle.

fetch_data.py — Téléchargement API NASA POWER
import requests
import pandas as pd
import os, time

VILLES = {
    "Ouagadougou":    {"lat": 12.36, "lon": -1.53},
    "Bobo-Dioulasso": {"lat": 11.18, "lon": -4.30},
    "Dori":           {"lat": 14.03, "lon": -0.03},
    "Fada N'Gourma":  {"lat": 12.06, "lon":  0.35},
    "Gaoua":          {"lat": 10.33, "lon": -3.18},
    "Dedougou":       {"lat": 12.46, "lon": -3.46},
}
PARAMETRES = ["PRECTOTCORR", "T2M_MAX", "T2M_MIN", "RH2M"]

def telecharger_ville(nom, lat, lon):
    url = "https://power.larc.nasa.gov/api/temporal/daily/point"
    params = {
        "parameters": ",".join(PARAMETRES),
        "community": "AG", "longitude": lon, "latitude": lat,
        "start": "19900101", "end": "20231231", "format": "JSON",
    }
    r = requests.get(url, params=params, timeout=90)
    r.raise_for_status()
    params_data = r.json()["properties"]["parameter"]
    dates = pd.date_range("1990-01-01", "2023-12-31", freq="D")
    df = pd.DataFrame(index=dates)
    df.index.name = "date"
    for param, values in params_data.items():
        s = pd.Series(values)
        s.index = pd.to_datetime(s.index, format="%Y%m%d")
        df[param] = s
    df.replace(-999.0, pd.NA, inplace=True)
    df["ville"] = nom
    return df
analyse.py — Cumul annuel & anomalies
import pandas as pd
import matplotlib.pyplot as plt
import numpy as np

def graphique_cumul_annuel(df):
    annuel = (
        df.groupby(["ville", df.index.year])["PRECTOTCORR"]
        .sum().reset_index()
        .rename(columns={"date": "annee", "PRECTOTCORR": "cumul_mm"})
    )
    fig, ax = plt.subplots(figsize=(13, 6))
    for ville in annuel["ville"].unique():
        d = annuel[annuel["ville"] == ville]
        ax.plot(d["annee"], d["cumul_mm"], label=ville, linewidth=1.6)
        # Tendance linéaire
        z = np.polyfit(d["annee"], d["cumul_mm"], 1)
        p = np.poly1d(z)
        ax.plot(d["annee"], p(d["annee"]), linestyle="--", linewidth=0.9)
    ax.set_title("Cumul annuel des précipitations (1990-2023)")
    ax.legend()
    plt.savefig("graphiques/01_cumul_annuel.png", dpi=150)
    plt.show()

def graphique_anomalies(df):
    annuel = (
        df.groupby(["ville", df.index.year])["PRECTOTCORR"]
        .sum().reset_index()
        .rename(columns={"date": "annee", "PRECTOTCORR": "cumul_mm"})
    )
    moyennes = annuel.groupby("ville")["cumul_mm"].transform("mean")
    annuel["anomalie_pct"] = (annuel["cumul_mm"] - moyennes) / moyennes * 100
    national = annuel.groupby("annee")["anomalie_pct"].mean().reset_index()
    fig, ax = plt.subplots(figsize=(13, 5))
    colors = ["steelblue" if v >= 0 else "tomato" for v in national["anomalie_pct"]]
    ax.bar(national["annee"], national["anomalie_pct"], color=colors)
    ax.axhline(0, color="black", linewidth=0.8)
    plt.savefig("graphiques/05_anomalies_precipitations.png", dpi=150)
    plt.show()
debut_saison.py — Détection du début de la saison agricole
def debut_saison_pluies(pluie_serie):
    """
    Retourne le jour julien du début de la saison.
    Critère agronomique : cumul sur 3 jours >= 20 mm après le 1er avril.
    """
    for i in range(90, len(pluie_serie) - 2):
        cumul3j = pluie_serie.iloc[i:i+3].sum()
        if cumul3j >= 20:
            return pluie_serie.index[i].dayofyear
    return None

def graphique_debut_saison(df):
    resultats = []
    for ville in df["ville"].unique():
        df_v = df[df["ville"] == ville]["PRECTOTCORR"].dropna()
        for annee in range(1990, 2024):
            serie = df_v[df_v.index.year == annee]
            jour = debut_saison_pluies(serie)
            if jour:
                resultats.append({"ville": ville, "annee": annee, "jour_debut": jour})
    res = pd.DataFrame(resultats)
    # Tracer l'évolution du début de saison par ville
    fig, ax = plt.subplots(figsize=(13, 6))
    for ville in res["ville"].unique():
        d = res[res["ville"] == ville].sort_values("annee")
        ax.plot(d["annee"], d["jour_debut"], label=ville, marker="o", markersize=3)
    plt.savefig("graphiques/04_debut_saison_pluies.png", dpi=150)
    plt.show()

Conclusions

Cinq grands enseignements issus de l'analyse de 33 années de données météorologiques pour le Burkina Faso.

Conclusion 01
Transition climatique majeure
Le Burkina Faso est passé d'une longue période sèche (1990–2004) à une période nettement plus pluvieuse depuis 2008. Ce basculement est cohérent avec la ré-verdification partielle du Sahel ouest-africain observée à l'échelle régionale.
Conclusion 02
Réchauffement climatique documenté
Un réchauffement de +0,7 °C à +1,0 °C selon les villes a été mesuré en 33 ans. Bobo-Dioulasso présente la hausse la plus marquée (+1,0 °C). Ce réchauffement menace directement la productivité agricole, chaque degré supplémentaire réduisant les rendements du maïs de 5 à 10 %.
Conclusion 03
Gradient Nord-Sud marqué
La zone Sud (Bobo, Gaoua) reçoit 1 000–1 100 mm/an et présente des conditions favorables à l'agriculture diversifiée. Le Nord-Sahel (Dori) avec seulement 516 mm/an, des températures max dépassant 36 °C et une variabilité extrême, est très vulnérable aux chocs climatiques.
Conclusion 04
Recommandations pour les semis
D'après l'analyse de la climatologie mensuelle et du début de saison : fin mai au Sud (Bobo, Gaoua), mi-juin au Centre (Ouagadougou, Dedougou, Fada). Au Nord (Dori), le maïs est déconseillé — privilégier mil et sorgho avec un semis en juillet.
Conclusion 05
Variabilité : principal risque alimentaire
Les écarts interannuels de −29 % (2002) à +18 % (depuis 2008) autour de la moyenne nationale constituent le principal risque pour la sécurité alimentaire. Cette variabilité dépasse souvent la capacité d'adaptation des petits agriculteurs sans mécanismes d'assurance ou de stockage.
